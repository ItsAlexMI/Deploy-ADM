# 💾 Sistemas Operativos

<Alert>
  <AlertTitle>¿Qué es un sistema operativo?</AlertTitle>
  <AlertDescription>
    Es el software principal que gestiona el hardware y los recursos de la computadora, proporcionando servicios a los programas y al usuario.
  </AlertDescription>
</Alert>


## Funciones principales

  <TableHeader>
    <TableRow>
      <TableHead>Función</TableHead>
      <TableHead>Descripción</TableHead>

    </TableRow>
  </TableHeader>
  <TableBody>
    <TableRow>
      <TableCell>Gestión de Procesos</TableCell>
      <TableCell>Controla la ejecución de programas y la asignación de CPU.</TableCell>
      <TableCell>Multitarea en Windows, Linux.</TableCell>
    </TableRow>
    <TableRow>
      <TableCell>Gestión de Memoria</TableCell>
      <TableCell>Asigna y libera memoria, implementa memoria virtual.</TableCell>
      <TableCell>Swap, paginación, RAM virtual.</TableCell>
    </TableRow>
    <TableRow>
      <TableCell>Gestión de Archivos</TableCell>
      <TableCell>Organiza y protege archivos y directorios.</TableCell>
      <TableCell>Sistemas de archivos NTFS, ext4.</TableCell>
    </TableRow>
    <TableRow>
      <TableCell>Gestión de Dispositivos</TableCell>
      <TableCell>Controla el acceso a hardware y periféricos.</TableCell>
      <TableCell>Drivers de impresora, USB.</TableCell>
    </TableRow>
    <TableRow>
      <TableCell>Interfaz de Usuario</TableCell>
      <TableCell>Permite la interacción usuario-sistema.</TableCell>
      <TableCell>CLI (terminal), GUI (ventanas).</TableCell>
    </TableRow>
  </TableBody>
</Table>

## Tipos de sistemas operativos


<div className="flex flex-wrap gap-4 my-6">
  <Card className="w-64 flex flex-col items-center">

  </Card>
  <Card className="w-64 flex flex-col items-center">
    <CardTitle>Multitarea</CardTitle>
    <div className="text-center">Permite ejecutar varios programas simultáneamente.</div>
  </Card>
  <Card className="w-64 flex flex-col items-center">
    <CardTitle>Monousuario</CardTitle>
    <div className="text-center">Un solo usuario a la vez.</div>
  </Card>
  <Card className="w-64 flex flex-col items-center">
    <CardTitle>Multiusuario</CardTitle>
    <div className="text-center">Varios usuarios pueden usar el sistema.</div>
  </Card>
  <Card className="w-64 flex flex-col items-center">
    <CardTitle>De red</CardTitle>
    <div className="text-center">Gestiona recursos y usuarios en red.</div>
  </Card>
  <Card className="w-64 flex flex-col items-center">
    <CardTitle>Embebidos</CardTitle>
    <div className="text-center">Diseñados para dispositivos específicos (ej: microondas, routers).</div>
  </Card>
  <Card className="w-64 flex flex-col items-center">
    <CardTitle>Móviles</CardTitle>
    <div className="text-center">Para smartphones y tablets.</div>
  </Card>
  <Card className="w-64 flex flex-col items-center">
    <CardTitle>Tiempo real</CardTitle>
    <div className="text-center">Responden en tiempo limitado, usados en control industrial.</div>
  </Card>
  <Card className="w-64 flex flex-col items-center">
    <CardTitle>Distribuidos</CardTitle>
    <div className="text-center">Varios sistemas operativos colaboran en diferentes máquinas para una tarea común.</div>
  </Card>
</div>







## Tipos de sistemas operativos

- Monotarea vs. multitarea
- Monousuario vs. multiusuario
- De red, embebidos, móviles, de tiempo real

### Ejemplos populares

- Windows, Linux, macOS, Android, iOS
